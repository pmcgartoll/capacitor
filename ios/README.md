# iOS apps

Native iOS projects that live alongside the web prototypes in this repo.

| App | Description | Path |
| --- | --- | --- |
| **Hush (SwiftUI)** | White noise player with battery-aware volume reduction | [`WhiteNoise/`](./WhiteNoise/) |

Open the `.xcodeproj` on a Mac with Xcode to build and run on an iPhone.

There is also an agent-testable **web / hexagonal** build of Hush at [`../hush/`](../hush/)
with the same battery policy and Capacitor-ready native adapter ports.
